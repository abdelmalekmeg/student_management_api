# Simple CRUD API made with Express.js (UNFINISHED)

Made in orderer to learn express, the project features small student management system with the options to Add, Remove, Update, and List students.

### How to run: 

+ Install the project repository.
+ Navigate to the server/ directory:

  `cd server` (must be inside `student_management_api` directory)
   
+ Start the server:

  `node index.js`

+ Start the web page:

  `cd ../client`
  `npm run dev`
